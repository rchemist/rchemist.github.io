This is the API

findCreditByAccountNumber

크레딧 Account Number 를 통해 크레딧 정보를 조회

URL

http://localhost:8280/api/credit/accountNumber/{accountNumber}

HTTP Method

GET

Response Type

CreditDTO

Parameters

  • Path: accountNumber String

    조회할 크레딧 Account Number

  • Query: includeTransaction Boolean

    크레딧 트랜잭션 (사용 히스토리) 정보 포함 여부

findCreditByExternalId

크레딧 외부 연동 키를 통해 크레딧 정보를 조회

URL

http://localhost:8280/api/credit/externalId/{externalId}

HTTP Method

GET

Response Type

CreditDTO

Parameters

  • Path: externalId String

    조회할 크레딧 외부 연동 키

  • Query: includeTransaction Boolean

    크레딧 트랜잭션 (사용 히스토리) 정보 포함 여부
