findActiveSubCategories

카테고리의 활성화된 하위 카테고리 전체 조회

URL

http://localhost:8280/catalog/category/{categoryId}/active-sub-categories

HTTP Method

GET

Response Type

CategoriesWrapper

Parameters

  • Path: categoryId Long

    카테고리 시퀀스 ID

  • Query: limit Integer

    페이지 사이즈

  • Query: offset Integer

    페이지 번호
