This is the API

findFulfillmentGroupAddress

주문 시퀀스 ID 와 배송 그룹 시퀀스 ID 로 배송그룹을 조회 후 배송지 조회

URL

http://localhost:8280/shipping/{orderId}/{fulfillmentGroupId}/address

HTTP Method

GET

Response Type

AddressWrapper

Parameters

  • Path: orderId Long

    주문 시퀀스 ID

  • Path: fulfillmentGroupId Long

    배송 그룹 시퀀스 ID

findFulfillmentGroupsById

시퀀스 ID 를 통해 배송 그룹 조회

URL

http://localhost:8280/shipping/{fulfillmentGroupId}

HTTP Method

GET

Response Type

FulfillmentGroupWrapper

Parameters

  • Path: fulfillmentGroupId Long

    배송 그룹 시퀀스 ID

findFulfillmentGroupsForOrder

주문에 연결된 배송 그룹 조회

URL

http://localhost:8280/shipping/{orderId}/groups

HTTP Method

GET

Response Type

List

Parameters

  • Path: orderId Long

    주문 시퀀스 ID

findFulfillmentOptions

배송 타입에 따른 배송비 조회

Parameters
