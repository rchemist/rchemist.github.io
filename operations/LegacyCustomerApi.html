This is the API

getCreditByCustomer

고객 ID 를 통해 고객이 소유한 크레딧 정보를 조회

URL

http://localhost:8280/api/customer/credit/{customerId}

HTTP Method

GET

Response Type

CreditDTO

Parameters

  • Path: customerId Long

    조회할 고객 ID

  • Query: includeTransaction Boolean

    크레딧 트랜잭션 (사용 히스토리) 정보 포함 여부

getCustomerById

고객 ID 를 통해 고객을 조회

URL

http://localhost:8280/api/customer/{id}

HTTP Method

GET

Response Type

CustomerDTO

Parameters

  • Path: id Long

    조회할 고객 ID
