findInventoriesByLocationId

재고 위치 ID를 통해 인벤토리 조회

URL

http://localhost:8280/inventory

HTTP Method

GET

Response Type

List

Parameters

  • Query: locationId Long

    재고 위치 시퀀스 ID

  • Query: offset Integer

    페이지 번호

  • Query: limit Integer

    페이지 사이즈

findInventoryById

시퀀스 ID를 통해 재고 조회

URL

http://localhost:8280/inventory/{inventoryId}

HTTP Method

GET

Response Type

InventoryWrapper

Parameters

  • Path: inventoryId Long

    재고 시퀀스 ID

updateInventory

재고 수정

URL

http://localhost:8280/inventory/{inventoryId}/quantity

HTTP Method

PUT

Response Type

InventoryWrapper

Parameters

  • Path: inventoryId Long

    재고 시퀀스 ID

  • Query: quantity Integer

    재고 수량

    재고 제한 수량
