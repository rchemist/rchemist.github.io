강력한 전자상거래 서비스

  • 상품 별 판매 정책 설정

    상품 별로 1인당 판매 수량, 1회당 판매 수량 제한
    특정 사용자에게만 판매하거나, 특정 결제 수단으로만 판매 가능

강력한 전자상거래 서비스

  • 다양한 프로모션 지원

    특정 조건에서 상품의 가격을 할인하거나 사은품 제공
    주문 총액 할인, 주문 상품 별 할인, 배송비 할인, BOGO, Tiered Price, …

  • Rule base 프로모션

    강력한 Rule 엔진을 이용해 고객, 주문, 주문 상품, 배송 그룹, URL, 시간 조건 세부 설정
    관리도구를 이용해 무정지 실시간 프로모션 지원

  • 적용 정책 세부 설정

    주문 당 적용 횟수, 1인 당 적용 횟수, 상품 당 적용 횟수, 주문 취소 시 오퍼 복구 여부
    오퍼를 중첩해 적용할 수 있는지 여부 등 세부 설정 가능

강력한 전자상거래 서비스

어느 상품이라도, 어떤 조건으로나, 어떤 방법으로든 판매할 수 있습니다.

  • 다양한 배송방식 지원

    상품의 배송방식에 따라 일반 주문, 픽업 주문, 바우처 주문 지원
    지정일 배송, 구매 예약 등 다양한 방식의 풀필먼트 서비스 제공

  • 통합 주문/장바구니

    서로 다른 사업자의 배송 위치에서 배송되거나, 배송 방식이 다른 주문 상품들을 하나의 주문으로 통합 관리

  • 클레임

    주문 전체 취소, 부분 취소, 전체 반품, 부분 반품 기능 제공
    클레임 사유 별로 설정된 소비자 귀책 여부에 따라 배송비나 추가 수수료 부과 가능

서비스 통합을 위한 RESTful API

RESTful API 를 이용해 쉽게 서비스를 통합, 확장할 수 있습니다.

  • Headless Commerce via RESTful API

  • Headless Commerce via RESTful API

    특정 Front Site 가 아니라 모든 서비스와 통합 가능
    ERP, Chatbot, Social commerce 등 다양한 방식으로 연동

  • 편리하고 강력한 API 보안

    관리도구를 이용해 API 권한 관리
    OAuth2 인증으로 클라이언트 개발 부담 최소화

  • 주문 제한

내장 CMS로 무정지 Front 관리

  • 디자인 테마

    사이트 프론트 템플릿 관리 기능

  • UI 컴포넌트 관리

    메뉴, 리디렉션, 페이지, 배너, 팝업, 정적 리소스(이미지) 등 관리
    사이트 별 또는 통합 설정 가능

  • 콘텐츠 캐싱

    CMS 콘텐츠를 캐싱하여 높은 성능 보장
    콘텐츠 변경이 감지되면 자동 캐시 만료

개발 없이 도메인 엔티티 확장

관리도구에서 실시간으로 엔티티를 확장하거나 새로 만들 수 있습니다.

  • 커스텀 필드

    기존 엔티티에 동적으로 필드를 추가

  • 고성능 동적 콘텐츠

    커스텀 엔티티와 커스텀 필드의 성능을 보장하기 위해 다양한 방식의 캐싱 제공
    추후 NoSQL 데이터 저장소와 통합해 더 강력한 성능과 자유로운 설정 기능 제공 예정

  • 웹훅 연동 내장 서비스

  • 간편 결제

  • 사용자 정의 옵션
